Fun in Function Fields

Author

Shyam Ravishankar

Published

May 26, 2025

After an unfortunately long hiatus, I’m back to blogging. Hopefully I can be more consistent with this going forward. The topic for this post is function field arithmetic, which more or less is arithmetic (i.e. number theory) in the setting of function fields. The guiding principle here is that the ring \(\mathbf{Z}\) (the base setting for number theory) is highly similar to the polynomial rings \(\mathbf{F}_p[t]\) where \(\mathbf{F}_p\) is the finite field with \(p\) elements. Indeed, both rings are Euclidean domains and their nontrivial quotients are finite. With this as our starting point, we can quickly see that classical constructions and theorems from elementary, analytic, and algebraic number theory have natural analogues when we work over \(\mathbf{F}_p[t]\). Not only this, but many well-known difficult and open problems from number theory are easily resolved in the function field setting! The theories over number fields and function fields are so strikingly similar that we even have developments for number fields inspired by the function field case: most notably the field of Iwasawa theory. Despite the many similarities between “classical” number theory and function field arithmetic, these are by and large just analogies; there are still key differences that we encounter between the number field setting and the function field setting. Even those theorems which can be uniformly stated for all global fields have to be proven separately with different methods.

New tools: What makes function field arithmetic easier?

I mentioned that things often get easier when we translate number theory to the function field setting. Here are a few reasons why (and when) this happens:

  • Quotients of characteristic \(p\) rings are all characteristic \(p\), hence easier to gain control of.
  • Characteristic \(p\) rings have the Frobenius endomorphism \(x \mapsto x^p\).
  • The arithmetic of function fields is the geometry of algebraic curves over \(\mathbf{F}_q\), so this gives us access to tools from algebraic geometry. One of the most fundametal tools we obtain is the Riemann–Roch theorem.
  • Zeta functions are rational functions, hence much simpler to study. In particular the Riemann hypothesis is true for these zeta functions.

The third and fourth points essentially encapsulate what I consider the most beautiful aspect of function field arithmetic. Concepts like divisors, the Picard group, and the Riemann–Roch theorem from algebraic geometry have purely algebraic formulations in the language of function fields, and one can understand these from either the algebraic or the geometric aspect1. What’s more, these constructions are very closely related to primes and the ideal and idèle class groups from algebraic number theory, so we might view function field arithmetic as a sort of “meeting point” between algebraic geometry and algebraic number theory.

For the benefit of the reader familiar with algebraic geometry, I’d like to elaborate on the identification between curves2 and function fields. Readers unfamiliar with algebraic geometry may gladly skip the rest of this section, as it is not at all necessary to understand what I’ll discuss later. Formally, over any field \(k\), we have an equivalence of categories

\[\left\{\underset{\text{with dominant morphisms}}{\text{proper normal curves}/k}\right\} \quad \iff \quad \left\{\underset{\text{with transcendence degree } 1}{\text{field extensions }K/k}\right\}.\] This equivalence takes a curve \(C/k\) to its function field \(k(C) = \kappa(\eta) = \mathcal{O}_{C,\eta}\) and a function field \(K/k\) to the adic spectrum \(\operatorname{Spa}(K,k)\), the points of which are valuations on \(K\) which are nonnegative on \(k\). In this way, our association of a function field \(K\) to its curve \(C\) identifies the (closed) points of \(C\) with (nontrivial) valuations of \(K\). This identification will be very apparent when we formulate divisors, the Picard group, and the Riemann–Roch theorem for function fields. You may also similarly notice a strong resemblence between the geometric and arithmetic definitions of the zeta function of a curve and its function field.

Elementary arithmetic: the ring \(\mathbf{F}_q[T]\)

Just like number theory starts with the integer ring \(\mathbf{Z}\), the story of function field arithmetic starts with the polynomial ring \(\mathbf{F}_q[T]\) over the finite field with \(q = p^r\) elements. This ring is like \(\mathbf{Z}\) in several regards:

  • It is a Euclidean domain, so for \(f,g \in \mathbf{F}_q[T]\) we can write \(g = fh + j\), where \(h,j \in \mathbf{F}_q[T]\) and the degree \(\deg j < \deg f\)3.
  • As a consequence of being a Euclidean domain, \(\mathbf{F}_q[T]\) is a unique factorization domain: given \(f \in \mathbf{F}_q[T]\) we can write \(f = \alpha P_1^{e_1}\cdots P_s^{e_s}\) where \(\alpha \in \mathbf{F}_q\), each \(P_i\) is an irreducible polynomial in \(\mathbf{F}_q[T]\) with leading coefficient \(1\) (i.e. it is monic), each \(e_i\) is a nonnegative integer, \(P_i \neq P_j\) whenever \(i \neq j\), and this factorization is unique up to rearranging the \(P_i^{e_i}\) terms.
  • If \(f \in \mathbf{F}_q[T]\) is nonzero then the quotient ring \(\mathbf{F}_q[T]/(f)\) is finite. In fact it has cardinality \(q^{\deg f}\).

The second bullet is an incarnation of the Fundamental Theorem of Arithmetic in function field land, so already we have our first major theorem. Unlike \(\mathbf{Z}\), we don’t have an ordering on \(\mathbf{F}_q[T]\), but we often take monic polynomials to serve the role that positive integers serve in classical number theory. Hence we’ll call a polynomial \(P \in \mathbf{F}_q[T]\) prime if it is monic and irreducible. Another useful tool from number theory is the Chinese Remainder Theorem, which lets us essentially reduce problems modulo \(n\) to considering information modulo prime powers. The same kind of statement holds for \(\mathbf{F}_q[T]\) as well4: If \(f,g \in \mathbf{F}_q[T]\) are coprime (have no prime factors in common), then there is an isomorphism of rings

\[\mathbf{F}_q[T]/(fg) \cong \mathbf{F}_q[T]/(f) \times \mathbf{F}_q[T]/(g).\]
