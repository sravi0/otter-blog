In this post I will discuss the existence and uniqueness of Haar measure, along with some of its properties.

Definition and Properties of Haar measure

Throughout this post, \(G\) will be a locally compact Hausdorff topological group and \(\mathcal{B}\) the \(\sigma\)-algebra of Borel sets on \(G\). A Radon measure on \(G\) is called a left Haar measure if it is left translation invariant i.e. \(\mu(xA) = \mu(A)\) for all \(x\in G\) and \(A \in \mathcal{B}\). Similarly, \(\mu\) is a right Haar measure if it is right translation invariant i.e. \(\mu(Ax) = \mu(A)\) for all \(A \in \mathcal{B}\) and \(x\in G\). The following proposition shows that existence of a left Haar measure on \(G\) is equivalent to the existence of a right Haar measure, and that translation invariance of a measure is equivalent to translation invariance of integrals.

Proposition 1 Let \(\mu\) be a Radon measure on \(G\). Then the following are equivalent:

  1. \(\mu\) is a left Haar measure;
  2. The measure \(\hat{\mu}\) defined by \(\hat{\mu}(A) = \mu(A^{-1})\) is a right Haar measure;
  3. \[ \int_G L_x f \, d\mu = \int_G f \, d\mu \quad \text{for all } f\in C_c(G) \text{ and } x \in G. \]

Proof. i. \(\iff\) ii. We first note that \(\mu\) is a Radon measure if and only if \(\hat{\mu}\) is a Radon measure. Indeed, since inversion is a homeomorphism we have that \(E\) is open (resp. compact, Borel) if and only if the same is true of \(E^{-1}\). Further, inversion preserves containment so the finiteness and regularity conditions follow. Now note that by the definition of \(\hat{\mu}\) we have the equivalence

\[ \hat{\mu}(Ex) = \hat{\mu}(E) \iff \mu(E^{-1}) = \mu(x^{-1}E^{-1}) \]

so it follows that \(\mu\) is a left Haar measure if and only if \(\hat{\mu}\) is a right Haar measure.

i. \(\implies\) iii. First suppose that \(\mu\) is a left Haar measure on \(G\). For a simple function \(s = \sum_{k=1}^n \alpha_k \mathbf{1}_{E_k}\) we have \(L_x s = \sum_{k=1}^n \alpha_k \mathbf{1}_{xE_k}\). Then

\[ \int_G L_x s \, d\mu = \sum_{k=1}^n \alpha_k \mu(xE_k) = \sum_{k=1}^n \alpha_k \mu(E_k) = \int_G s \, d\mu \]

so the assertion is true for simple functions. Now note that \(s \leq f\) if and only if \(L_x s \leq L_x f\), so it follows that

\[ \int_G L_x f \, d\mu = \int_G f \, d\mu \]

for all nonnegative measurable functions \(f\). For complex-valued \(f\), we write \(f\) as \((g_+ - g_-) + i(h_+ - h_-)\) where \(g_+, g_-, h_+, h_-\) are nonnegative measurable functions. The result then follows from linearity of the integral as well as the fact that \(L_x (f_1 + f_2) = L_x f_1 + L_x f_2\).

i. \(\impliedby\) iii. By the proof of the Riesz Representation Theorem, we can recover \(\mu\) on open sets by

\[ \mu(U) = \sup\left\{ \int_G f \, d\mu \colon f \in C_c(G), \, 0 \leq f \leq 1, \, \mathrm{supp} f \subset U \right\}. \]

From this we can immediately see that if \(\int_G L_x f \, d\mu = \int_G f \, d\mu\) then \(\mu\) must be left translation invariant on all open sets. Since \(\mu\) is a Radon measure and thus outer regular, we can extend left translation invariance to all Borel sets using properties of \(L_x\).

\(\square\)

Proposition 2 Suppose now that \(\mu\) is a nonzero left Haar measure on \(G\); then the following also hold:

  1. \(\mu\) is positive on all nonempty open subsets of \(G\);
  2. If \(f \in C_c(G)\) is nonzero with range contained in \([0,\infty]\), then \(\int_G f \, d\mu > 0\);
  3. \(\mu(G)\) is finite if and only if \(G\) is compact.

Proof. i. Since \(\mu\) is nonzero, we must have \(\mu(G) > 0\); by inner regularity there exists a compact set \(K \subset G\) such that \(\mu(K) > 0\). Let \(U \subset G\) be open; the translates \(\{ gU \colon g \in G \}\) cover \(G\), so they must cover \(K\) as well. Since \(K\) is compact there exist \(g_1,\ldots, g_n \in G\) such that

\[ K \subset \bigcup_{j=1}^n g_jU. \]

Then since the translates of \(U\) are disjoint we have

\[ 0 < \mu(K) \leq \mu\left(\bigcup_{j=1}^n g_jU \right) = \sum_{j=1}^n \mu(g_jU) = n\mu(U); \]

it thus follows that \(\mu(U) > 0\).

ii. Let \(f \in C_c(G)\) be nonzero with nonnegative range; then we know that there exists \(\epsilon > 0\) such that \(\epsilon \in f(G)\). Define \(U \coloneqq f^{-1}\left(\left(\frac{\epsilon}{2}, \frac{3\epsilon}{2}\right)\right)\), so that \(U\) is nonempty and \(f > \frac{\epsilon}{2}\) on \(U\). Then since \(\mu(U) > 0\) by i., we have

\[ \int_G f \, d\mu = \int_{U} f \, d\mu + \int_{G \setminus U} f \, d\mu > \int_U \frac{\epsilon}{2} \, d\mu = \frac{\epsilon}{2}\mu(U) > 0. \]

iii. If \(G\) is compact then \(\mu(G)\) must be finite since \(\mu\) is a Radon measure. Conversely, suppose \(G\) is not compact; by inner regularity there exists a compact \(K \subset G\) such that \(\mu(K) > 0\). Define \(g_1 := 1\) to be the identity of \(G\); for any \(g_1,\ldots, g_n \in G\) we must have that

\[ \bigcup_{i=1}^n g_iKK^{-1} \neq G \]

since \(K^{-1}\) and thus \(KK^{-1}\) is compact. Thus there exists \(g_{n+1} \in G \setminus \bigcup_{i=1}^n g_iKK^{-1}\); we claim that by design \(g_{n+1}K\) is disjoint from \(g_iK\) for all \(i = 1, \ldots, n\). Indeed, any \(x \in g_{n+1}K\) can be written as \(x = g_{n+1}k_1\) for some \(k_1 \in K\). If \(x \in g_iK\) for some \(i\leq n\) then we can similarly write \(x = g_i k_2\) for some \(k_2 \in K\). Then \(xk_1^{-1} = g_{n+1} = g_i k_2k_1^{-1} \in g_i KK^{-1}\), contradicting the construction of \(g_{n+1}\). We thus have a sequence \((g_i)\) in \(G\) such that the sets \(g_iK\) are pairwise disjoint. Countable additivity of \(\mu\) then gives that

\[ \mu(G) \geq \sum_{i=1}^\infty \mu(g_iK) = \sum_{i=1}^\infty \mu(K) = \infty. \]

\(\square\)

Uniqueness of Haar Measure

Before proving existence of Haar measure for locally compact groups, we prove that Haar measure is unique up to scaling. For abelian groups this is particularly easy:

Proposition 3 (Uniqueness of Haar Measure for Abelian Groups) Let \(\mu\) and \(\nu\) be nonzero Haar measures on a locally compact abelian group \(G\). Then there exists some nonnegative constant \(\alpha\) such that \(\nu(E) = \alpha\mu(E)\) for all \(E \in \mathcal{B}\).

Proof. We will prove that \(\int_G f \, d\nu = \alpha \int_G f \, d\mu\) for all \(f \in C_c(G)\); the result will follow from the Riesz Representation Theorem.

Fix a nonzero \(g \in C_c(G)\); by the extreme value theorem, \(|g|\) is universally bounded by some constant \(C\). From this it follows that \(\int_G |g| \, d\mu \leq C\mu(\mathrm{supp} \, g) < \infty\), and by Proposition 2 the integral is nonzero. Thus replacing \(g\) with \(\left(\int_G |g| \, d\mu\right)^{-1}|g|\), we may assume without loss of generality that \(g \geq 0\) and \(\int_G g \, d\mu = 1\). Define \(\alpha\) by

\[ \alpha \coloneqq \int_G g(-x) \, d\nu(x) \]

and let \(f \in C_c(G)\). We then have

\[\begin{align} \int_G f(x) \, d\nu(x) &= \left(\int_G g(y) \, d\mu(y) \right) \int_G f(x) \, d\nu(x) && \text{since } \int_G g \, d\mu = 1\\ &= \int_G \int_G g(y)f(x) \, d\nu(x) d\mu(y)\\ &= \int_G \int_G g(y)f(y+x) \, d\nu(x) d\mu(y) && \text{since the Haar integral is translation invariant}\\ &= \int_G \int_G g(y) f(y+x) \, d\mu(y) d\nu(x) && \text{since } g(y)f(x+y) \in C_c(G \times G), \text{ Fubini's theorem applies}\\ &= \int_G \int_G g(-x+y) f(y) \, d\mu(y) d\nu(x) && \text{since the Haar integral is translation invariant and } G \text{ is abelian}\\ &= \int_G \int_G g(-x+y) f(y) \, d\nu(x) d\mu(y) && \text{since } g(-x+y)f(y) \in C_c(G \times G), \text{ Fubini's theorem applies}\\ &= \int_G \int_G g(-x) f(y) \, d\nu(x) d\mu(y) && \text{since } G \text{ is abelian and the Haar integral is translation invariant}\\ &= \int_G g(-x) \, d\nu(x) \int_G f(y) \, d\mu(y)\\ &= \alpha \int_G f \, d\mu && \text{by definition of } \alpha. \end{align}\]

\(\square\)

For arbitrary locally compact groups this takes a bit more work, though we will still use the idea of proving \(\int_G f \, d\nu = \alpha \int_G f \, d\mu\) for all \(f \in C_c(G)\).

Theorem 1 (Uniqueness of Haar Measure) Let \(\mu\) and \(\nu\) be nonzero left Haar measures on a locally compact group \(G\). Then there exists some nonnegative constant \(\alpha\) such that \(\nu(E) = \alpha\mu(E)\) for all \(E \in \mathcal{B}\).

Proof. As in the abelian case, let \(g \in C_c(G)\) such that \(g \geq 0\) and \(\int_G g \, d\mu = 1\), and define

\[ \alpha \coloneqq \int_G g \, d\nu. \]

Let \(f \in C_c(G)\) and define \(h \colon G \times G \to \mathbf{C}\) by

\[ h(x,y) \coloneqq \frac{f(x)g(yx)}{\int g(tx) \, d\nu(t)}. \]

By Proposition 2, the integral in the denominator is nonzero (though note that it does depend on \(x\) since the Haar integral is only left invariant and may not be right invariant). Since \(G\) is a topological group, the map \(G \times G \to \mathbf{C}\) given by \((t,x) \mapsto g(tx)\) is continuous, so by Fubini’s theorem the map \(x \mapsto \int_G g(tx) \, d\nu(t)\) is also continuous. Thus \(h\) is continuous as the product of continous functions. Now let \((x,y) \in \operatorname{supp}h\); then we must have \(f(x) \neq 0\) and \(g(yx) \neq 0\) so \(x \in \operatorname{supp}f\) and \(yx \in \operatorname{supp}g\). From this we can see that \(\operatorname{supp}h \subset \operatorname{supp}f \times \operatorname{supp}g(\operatorname{supp}f)^{-1}\) so \(h\) has compact support. We then have

\[\begin{align} \int_G \int_G h(x,y) \, d\nu(y) d\mu(x) &= \int_G \int_G h(x,y) \, d\mu(x) d\nu(y) && \text{by Fubini's theorem}\\ &= \int_G \int_G h(y^{-1}x, y) \, d\mu(x) d\nu(y) && \text{since the Haar integral is left translation invariant}\\ &= \int_G \int_G h(y^{-1}x, y) \, d\nu(y) d\mu(x) && \text{by Fubini's theorem}\\ &= \int_G \int_G h(y^{-1}, xy) \, d\nu(y) d\mu(x) && \text{since the Haar integral is left translation invariant}. \end{align}\]

Now with \(h\) as defined before, we have

\[ \int_G \int_G h(x,y) \, d\nu(y) d\mu(x) = \int_G \frac{f(x)}{\int g(tx) \, d\nu(t)} \int_G g(yx) \, d\nu(y) d\mu(x) = \int_G f(x) \, d\mu(x) \]

and

\[ \int_G \int_G h(y^{-1}, xy) \, d\nu(y) d\mu(x) = \int_G g(x) \int_G \frac{f(y^{-1})}{\int_G g(ty^{-1}) \, d\nu(t)} \, d\nu(y) d\mu(x) = \int_G g(x) \, d\mu(x) \int_G \frac{f(y^{-1})}{\int_G g(ty^{-1}) \, d\nu(t)} \, d\nu(y). \]

Thus the ratio \(\frac{\int_G f \, d\mu}{\int_G g \, d\mu}\) depends only on \(f\) and \(g\), and has no dependence on the measure \(\mu\). Thus we have

\[ \frac{\int_G f \, d\mu}{\int_G g \, d\mu} = \frac{\int_G f \, d\nu}{\int_G g \, d\nu} \]

which we can rearrange to obtain

\[ \int_G f\, d\nu = \int_G f\, d\nu \int_G g \, d\mu = \int_G g\, d\nu \int_G f\, d\mu = \alpha \int_G f \, d\mu \]

as desired.

\(\square\)

Existence of Haar Measure

The remainder of this post will be devoted to proving that every locally compact group \(G\) admits a left Haar measure. As demonstrated in Proposition 1, this is equivalent to the existence of a right Haar measure on \(G\), as well as the existence of a positive, left-translation invariant functional on \(C_c(G)\). The construction I will follow is the one developed by Weil in (Weil 1965) and presented in (Cohn 2013). An alternative construction due to Cartan leverages the Riesz Representation Theorem to simultaneously prove existence and uniqueness while avoiding use of the axiom of choice (Weil’s construction requires use of Tychonoff’s Theorem) and is presented in (Hewitt and Ross 2012). There is also a shorter variant of Cartan’s argument in (Ramakrishnan and Valenza 1999) which does use Tychonoff’s Theorem.

The existence of a left Haar measure will be broken into a number of steps:

  1. Define an “index” \([A:B]\) for \(A,B \subset G\);
  2. Use the index to define a family of “pseudo-contents” on compact sets;
  3. Define the “Haar content” on compact sets as a “limit” of the pseudo-contents;
  4. Define Haar outer measure on open sets via inner regularity;
  5. Extend Haar outer measure to all sets via outer regularity;
  6. Show the outer measure forms a measure when restricted to Borel sets;
  7. Prove the Haar measure is left translation invariant;
  8. Prove the Haar measure is nonzero.

We now state and prove Haar’s theorem:

Theorem 2 (Existence of Left Haar Measure) Let \(G\) be a locally compact Hausdorff topological group. There exists a nonzero left Haar measure \(\mu\) on \(G\).

Proof. We proceed in the steps described above.

Step i. Defining \([A:B]\)

Let \(A,B \subset G\); we define the index \([A:B]\) to be the smallest \(n \in \mathbf{N}\) such that there are \(n\) translates of \(B\) which cover \(A\). Note that \([A:B] = 0\) if and only if \(A = \varnothing\); if \(A\) cannot be covered by finitely many translates of \(B\), then we set \([A:B]=\infty\) (this case will not concern us going forward). When \(K \subset G\) is compact and \(U \subset G\) is nonempty and open, we note that \([K:U]\) must be finite.

Step ii. Defining \(h_U\)

Fix a compact set \(K_0\) with nonempty interior; such a set must exist since \(G\) is locally compact. For compact \(K\) and nonempty open \(U\), we define the “pseudo-content”

\[ h_U(K) \coloneqq \frac{[K:U]}{[K_0:U]}. \]

Lemma 1 (Properties of \(h_U\)) The function \(h_U\) satisfies the following properties:

  1. \(0 \leq h_U \leq [K:K_0]\);

Step iii. Defining \(h\) on compact sets

Let \(X \coloneqq \prod_{K \subset G} \left[0, [K:K_0] \right]\), where the product is taken over all compact subsets of \(G\). Due to Lemma 1 i., we can consider the function \(h_U\) as a point in \(X\) for each \(U\). For each open neighborhood \(V \subset G\) of the identity, we define the set

\[ M(V) \coloneqq \overline{\left\{ h_U \colon U \subset V \right\}}; \]

we claim that the collection \(\{M(V)\}\) satisfies the finite intersection property. Indeed, for \(V_1,\ldots, V_n \subset G\) open, the intersection \(\bigcap_{i=1}^n V_i\) is open and contains \(e\), so \(h_{\bigcap_{i=1}^n V_i} \in \bigcap_{i=1}^n M(V_i)\). By Tychonoff’s theorem, \(X\) is compact, so the intersection \(\bigcap_{V \subset G} M(V)\) is nonempty. We then let \(h\) be some element of this intersection.

Lemma 2 (Properties of \(h\)) The function \(h\) satisfies the following properties on compact sets:

  1. If \(A \subset K\) then \(h(A) \leq h(K)\);

Step iv. Defining \(\mu\) on open sets

For \(U \subset G\) open, we define

\[ \mu(U) \coloneqq \sup\left\{ h(K) \colon K \subset U, \, K \text{ compact} \right\}. \]

If \(U\) is open and compact this coincides with \(h\). Indeed, if \(K \subset U\) is compact then \(\mu(K) \leq \mu(U)\) by Lemma 2 i.; further, since \(U\) is compact, \(\mu(U)\) is achieved so it must be the supremum of \(\{\mu(K) \colon K\subset U\}\).

Lemma 3 (Properties of \(\mu\)) The function \(\mu\) satisfies the following properties on open sets:

  1. If \(U \subset V\) then \(\mu(U) \leq \mu(V)\);

Step v. Extending \(\mu\) to all sets

For any \(A \subset G\), we define

\[ \mu(A) \coloneqq \inf\left\{ \mu(U) \colon A \subset U, \, U \text{ open} \right\}. \]

Once again, this agrees with the previous definition on open sets, since \(\mu(A) \leq \mu(U)\) for open sets \(A \subset U\) (Lemma 2) and \(\mu(A)\) is achieved in the infimum of \(\{\mu(U) \colon A \subset U\}\) when \(A\) is open1.

Step vi. Restricting \(\mu\) to Borel sets

We now show that when \(\mu\) is restricted to the Borel sets of \(G\) it forms a measure. To do this, we use Carathéodory’s criterion and check that the open sets \(U \subset G\) satisfy

\[ \mu(A) = \mu(A \cap U) + \mu(A \cap U^c) \]

for all \(A \subset G\).
